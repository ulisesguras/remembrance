The problem

Agents are powerful, but they start from zero every time.

The limitation isn't storage — it's architecture. Memory is usually treated as a flat log that grows until it's truncated. No structure, no prioritization, no learning.

No continuity

Every session starts fresh. Agents can't carry forward what they learned.

Rebuilt every time

Developers re-implement memory and state logic from scratch in every project.

No improvement

Without persistent memory, agents can't learn from past experience.

No shared knowledge

In multi-agent systems, each agent operates in isolation — nothing is shared.

The architecture

Nine layers across three cognitive dimensions.

A synthesis of established cognitive-science memory models, applied to agent architecture. Hover a layer to trace it.

By duration
01sensoryRaw inputs with TTL-based decay — what just arrived.
02workingActive scratchpad, priority-evicted — what's being reasoned about now.
03episodicAppend-only event log — what happened and when.
04semanticFact store with pluggable vector backend — knowledge that grows.
By function
05proceduralNamed skill registry — how to do things.
06strategicSelf-reflection and strategy tracking — how to get better over time.
By scope
07emotionalAffect signals per subject — risk calibration.
08prospectiveFuture intentions, triggered by time or events.
09collectiveShared knowledge across an agent fleet — what the team knows together.

How it works

One base class. All nine layers.

Instantiate BaseAgent and every layer is wired and working. No configuration, no manual layer management.

agent.py
from remembrance import BaseAgent, AgentConfig
from remembrance.memory.collective import CollectiveMemory

agent = BaseAgent(
    config=AgentConfig(name="agent-01", role="assistant", domain="support"),
    collective=CollectiveMemory(),
)

# all nine layers initialized automatically:
# sensory · working · episodic · semantic · procedural
# strategic · emotional · prospective · collective

agent.perceive("user_message", "the contract has a problematic clause")
agent.learn("force_majeure", "...", tags=["legal"])
agent.reflect(decision="deductive analysis", outcome="correct")
